# 🎬 CineSearch

A modern movie discovery web application built using **React**, designed to provide an intuitive browsing experience with dynamic search, filtering, and sorting capabilities.

---

## ✨ Features

- 🔍 **Real-time Search** – Instantly search movies by title
- ⭐ **Rating Filter** – Filter movies based on user ratings
- 📊 **Sorting Options** – Sort by rating or release date
- 🌗 **Dark/Light Mode** – Seamless theme switching using CSS variables
- 🎯 **Responsive UI** – Clean and adaptive layout across devices
- 🧩 **Reusable Components** – Modular and maintainable code structure

---

## 🏗️ Architecture

This project follows a **clean and scalable frontend architecture**:

- **Component-Based Design** – Reusable and isolated UI components
- **Service Layer Pattern** – Abstracted data fetching logic
- **Mock API Simulation** – Structured data mimicking real-world API responses
- **Separation of Concerns** – Clear distinction between UI, logic, and data

---

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Languages:** JavaScript (ES6+), HTML5, CSS3
- **Styling:** Custom CSS, CSS Variables (Theming)
- **Utilities:** Lodash
- **Concepts:** Component Architecture, State Management (Hooks), Service Layer Abstraction

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/cine-search.git
cd cine-search
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
